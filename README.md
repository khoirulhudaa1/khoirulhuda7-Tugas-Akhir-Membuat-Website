# 🌊 Pesona Kota Pekalongan

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=111111)
![Responsive](https://img.shields.io/badge/Responsive-0F8B8D?style=for-the-badge&logo=googlechrome&logoColor=white)

Website informasi sederhana tentang **Kota Pekalongan** yang menampilkan sejarah, geografis, wisata, kuliner, dan profil singkat kota dengan tampilan modern, responsif, serta mode gelap.

---

## ✨ Preview

Halaman ini dirancang dengan gaya card modern, hero image, navigasi sticky, dan layout responsif agar nyaman dibuka dari desktop maupun perangkat mobile.

![Preview Batik Pekalongan](assets/image/kota-batik.jpg)

---

## 🚀 Fitur Utama

| Icon | Fitur | Deskripsi |
| --- | --- | --- |
| 🧭 | Navigasi Sticky | Menu tetap mudah diakses saat halaman digulir. |
| 🌙 | Dark Mode | Tombol untuk mengganti tampilan terang dan gelap. |
| 📱 | Responsive Layout | Tampilan menyesuaikan desktop, tablet, dan mobile. |
| 🖼️ | Featured Image | Gambar artikel dibuat rapi dengan ukuran konsisten. |
| 🍲 | Grid Kuliner | Informasi kuliner ditampilkan dalam kartu yang tertata. |
| ⬆️ | Scroll to Top | Tombol cepat untuk kembali ke bagian atas halaman. |

---

## 🧩 Bagian Halaman

- **Beranda**: hero section dengan pengantar Kota Pekalongan.
- **Sejarah**: cerita singkat perkembangan kota dan identitas batik.
- **Geografis**: informasi letak dan karakter wilayah pesisir.
- **Wisata**: rekomendasi tempat dan aktivitas menarik.
- **Kuliner**: daftar makanan khas seperti tauto, sego megono, dan olahan laut.
- **Profil Kota**: ringkasan data Kota Pekalongan dalam sidebar.

---

## 🛠️ Teknologi

Proyek ini dibuat menggunakan teknologi dasar web:

- **HTML5** untuk struktur halaman semantik.
- **CSS3** untuk layout, responsivitas, kartu, warna, dan dark mode.
- **JavaScript Vanilla** untuk interaksi menu, tema, dan tombol scroll.

## ▶️ Cara Menjalankan

1. Buka folder proyek `web_pekalongan`.
2. Klik dua kali file `index.html`.
3. Website akan terbuka di browser.

Atau jalankan dengan server lokal:

```bash
python -m http.server 8080
```

Lalu buka:

```text
http://localhost:8080
```
